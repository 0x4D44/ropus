## Checkpoint Rules

After each significant batch:

1. Run `cargo +nightly llvm-cov --branch --json --summary-only --output-path target\llvm-cov-summary.json`.
2. Record the new totals.
3. Rank hotspots again by:
   - missed lines
   - missed branches
   - branch percentage
4. Stop pretending a stage worked if the slope has flattened.

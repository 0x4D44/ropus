# HLD: theta_rdo Stereo Encoding

## Problem

All 30 stereo non-silence encode tests fail because `theta_rdo` is hardcoded to `false` in `src/celt/bands.rs:1606`. This disables rate-distortion optimization for stereo band encoding, causing different theta quantization decisions than the C reference.

## Root Cause

In the C reference, stereo encoding at complexity >= 8 tries two theta roundings (up and down) for each band below the intensity stereo boundary, picks the one with lower perceptual distortion. This requires:
1. Saving encoder state before the first attempt
2. Running the band encoder twice
3. Measuring distortion after each
4. Restoring the winner's state

The Rust code always uses `theta_round = 0` (standard mid-point rounding), producing different theta values.

## Design

### Change 1: Enable theta_rdo and resynth

In `quant_all_bands` (`src/celt/bands.rs`), line ~1606:

```rust
// Before:
let theta_rdo = false;
let resynth = !encode || theta_rdo;

// After:
let theta_rdo = encode && y_.is_some() && !dual_stereo && complexity >= 8;
let resynth = !encode || theta_rdo;
```

This enables `resynth = true` during stereo encoding at high complexity, which in turn enables:
- Reconstructed signal write-back in `alg_quant`
- Norm buffer population for inter-band folding
- `lowband_offset` updates

### Change 2: Allocate save buffers

When `theta_rdo` is true, allocate temporary buffers for the two-pass save/restore. Add after the existing allocation block (~line 1610):

```rust
let n_max = if theta_rdo {
    (m.ebands[m.nb_ebands as usize] as i32 - m.ebands[m.nb_ebands as usize - 1] as i32) << lm
} else { 0 } as usize;
let mut x_save = vec![0i32; n_max];
let mut y_save = vec![0i32; n_max];
let mut x_save2 = vec![0i32; n_max];
let mut y_save2 = vec![0i32; n_max];
let mut norm_save2 = vec![0i32; n_max];
let mut bytes_save = vec![0u8; if theta_rdo { 1275 } else { 0 }];
```

### Change 3: The two-pass block

In the stereo non-dual-stereo band loop (currently around line 1857), replace the single `quant_band_stereo` call with:

```rust
if theta_rdo && i_band < intensity {
    let w = compute_channel_weights(band_e[iu], band_e[iu + nbe]);
    let cm = x_cm | y_cm;

    // Save state before pass 1
    let ec_snap1 = ec.snapshot();  // scalar encoder state
    let nstart = ec.range_bytes();
    let nend = ec.storage();
    let ctx_save = (/* clone relevant ctx fields */);
    x_save[..nu].copy_from_slice(&x[x_off..x_off + nu]);
    y_save[..nu].copy_from_slice(&y[y_off..y_off + nu]);

    // Pass 1: round down
    ctx.theta_round = -1;
    let (cm_out_1, _) = quant_band_stereo(/* ... */);
    let dist0 = mult16_32_q15(w[0], celt_inner_prod_norm_shift(&x_save, &x[x_off..], nu))
              + mult16_32_q15(w[1], celt_inner_prod_norm_shift(&y_save, &y[y_off..], nu));

    // Save pass-1 result
    let ec_snap2 = ec.snapshot();
    let ctx_save2 = (/* clone ctx */);
    x_save2[..nu].copy_from_slice(&x[x_off..x_off + nu]);
    y_save2[..nu].copy_from_slice(&y[y_off..y_off + nu]);
    if !last { norm_save2[..nu].copy_from_slice(&norm[norm_off..norm_off + nu]); }
    bytes_save[..nend - nstart].copy_from_slice(&ec.buffer()[nstart..nend]);

    // Restore pre-pass-1 state
    ec.restore(&ec_snap1);
    // restore ctx fields
    x[x_off..x_off + nu].copy_from_slice(&x_save[..nu]);
    y[y_off..y_off + nu].copy_from_slice(&y_save[..nu]);

    // Pass 2: round up
    ctx.theta_round = 1;
    let (cm_out_2, _) = quant_band_stereo(/* ... */);
    let dist1 = mult16_32_q15(w[0], celt_inner_prod_norm_shift(&x_save, &x[x_off..], nu))
              + mult16_32_q15(w[1], celt_inner_prod_norm_shift(&y_save, &y[y_off..], nu));

    // Pick winner
    if dist0 >= dist1 {
        // Pass 1 was better — restore its state
        ec.restore(&ec_snap2);
        // restore ctx_save2
        x[x_off..x_off + nu].copy_from_slice(&x_save2[..nu]);
        y[y_off..y_off + nu].copy_from_slice(&y_save2[..nu]);
        if !last { norm[norm_off..norm_off + nu].copy_from_slice(&norm_save2[..nu]); }
        ec.buffer_mut()[nstart..nend].copy_from_slice(&bytes_save[..nend - nstart]);
    }
    // else: pass 2 was better, state already in place
    (x_cm, y_cm) = (cm_out_winner);
} else {
    ctx.theta_round = 0;
    // existing single-pass quant_band_stereo call
}
```

### Change 4: Fix fold_start off-by-one

At line ~1707, the loop must pre-decrement before testing (matching C's `while(M*eBands[--fold_start]...)`):

```rust
// Before:
while big_m * m.ebands[fold_start - 1] as i32 > effective_lowband + norm_offset as i32 {
    fold_start -= 1;
    if fold_start == 0 { break; }
}

// After:
loop {
    fold_start -= 1;
    if !(big_m * m.ebands[fold_start] as i32 > effective_lowband + norm_offset as i32) { break; }
}
```

### Change 5: EcCoder trait extension

The `snapshot()` / `restore()` and buffer access methods are needed inside the generic `quant_all_bands<EC: EcCoder>` function. Add these to the `EcCoder` trait:

```rust
trait EcCoder {
    // existing methods...
    fn snapshot(&self) -> EcSnapshot { unimplemented!() }
    fn restore(&mut self, _snap: &EcSnapshot) { unimplemented!() }
    fn range_bytes(&self) -> usize { 0 }
    fn storage(&self) -> usize { 0 }
    fn buffer(&self) -> &[u8] { &[] }
    fn buffer_mut(&mut self) -> &mut [u8] { &mut [] }
}
```

Implement these on `RangeEncoder`. The `RangeDecoder` defaults are fine since theta_rdo is only active when `encode == true`.

## Testing

All 30 stereo non-silence encode tests should flip to PASS. No mono regressions expected since theta_rdo is false for mono (Y_ is None).

## Risks

1. **Buffer byte save/restore**: The C code saves raw encoder buffer bytes because struct-copy doesn't capture them. Must verify that `RangeEncoder::snapshot()` includes the buffer state. If not, manual byte save/restore is needed (as shown above).

2. **special_hybrid_folding re-call**: Between pass 1 and pass 2, when `i_band == start + 1`, C re-runs `special_hybrid_folding`. Must replicate this or the folding basis differs between passes.

3. **ctx fields**: The `band_ctx` struct has several fields (`remaining_bits`, `seed`, etc.) that must be saved/restored between passes. Need to identify all of them from the C `ctx_save = ctx` copy.

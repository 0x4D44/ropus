### T+1 — HLD reviews

**Bug 11 — APPROVED.** Missing low-bitrate SILK rate-capping block from C
`opus_encoder.c:2128-2148`. The block caps `maxInternalSampleRate` and
`desiredInternalSampleRate` when effective bitrate < 8000/7000 bps. Entirely
absent from Rust `encoder.rs:2006-2007`. Port omission, simple insert.

**Bug 12 — APPROVED WITH MODIFICATIONS.** Structural prefill ordering bug in
`silk_encode`: C does reset + set prefill flags BEFORE `silk_control_encoder`,
Rust does it AFTER. Correct diagnosis. Modifications:
- Don't switch to passing `&mut self.silk_mode` directly (keep clone)
- Don't mix in the `silk_SMLAWB` truncation fix (separate bug)
- Don't mix in `eprintln!` cleanup (separate commit)
- Must not regress commit 1500771's fixes (VAD warmup, DTX init, LBRR reset)

**Bug 13 — APPROVED.** Missing `silk_smulbb` i16 truncation in
`silk_NLSF_residual_dequant` at `common.rs:1241`. C uses `silk_SMULBB(out_Q10,
pred_coef)` which truncates `out_q10` to i16; Rust does a full i32 multiply.
One-line fix. Defer PLC wrapper fix.

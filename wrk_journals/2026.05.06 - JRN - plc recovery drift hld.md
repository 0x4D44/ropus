## Entry 2

Design choice: keep the existing exact first-288-sample regression and add a separate active bounded-drift test, rather than converting the ignored exact test.

That feels like the honest shape. Full-frame exact decode24 PLC parity is still unpaid debt, and deleting the ignored exact test would make that easier to forget. The active bounded test should fail if the known residual drift gets worse, but it must not pretend that residual drift is parity.

Emotional temperature: mildly wary. Bounded numeric oracles are useful but easy to abuse. The HLD wording has to be blunt that this is a debt-containing oracle, not a correctness victory lap.

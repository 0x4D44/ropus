## Key Insight: Devil's Advocate
A "devil's advocate" agent correctly predicted that the ±1 rounding was NOT from
individual math functions (all verified bit-exact) but from **constant values and
state initialization**. The Haar f32/f64 constant bug was the single biggest fix
(+31 tests). The lsb_depth initialization was second (+16 tests).
